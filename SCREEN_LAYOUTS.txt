================================================================================
DATA DISPLAYED BY SCREEN
================================================================================

OVERVIEW Screen:
  • Probe Label (Probe A, B, C, D)
  • Status (OK or ALARM)
  • Last Measurement Date (DD.MM.YYYY)
  • Current Temperature (X.X°C)
  • Page Number (Page X/Y)
  • Current Time (HH:MM:SS)
  • Alarm Priority Sorting (Alarms shown first)

SERVICE MENU Screen:
  • Menu Items (3 options)
  • No real-time data

SYSTEM INFO Screen:
  • Device ID (FIBER-001)
  • Current Date (DD.MM.YYYY)
  • Number of Connected Probes (count)
  • Power Level (0-100%)
  • Firmware Version (X.X.X)

SENSOR DETAILS Screen:
  • Probe Label
  • Current Value (Temperature in °C)
  • Last Measurement Date
  • Warning Thresholds (low/high)
  • Alarm Thresholds (low/high)
  • Warning Delay (minutes)
  • Alarm Delay (minutes)
  • Measurement Interval (minutes)
